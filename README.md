# n8n-nodes-zoho-assist

🚀 Full-featured Zoho Assist integration for n8n with session automation, recording downloads, and cloud storage pipeline support.

---

## ✨ Features

### 🔐 Authentication
- OAuth2 (Auto refresh supported)
- Multi Data Center support (India / US / EU)

### 🖥️ Core APIs
- Sessions (create, start, schedule)
- Devices (unattended access)
- Groups
- Reports
- Users

### 🎥 Recording Support
- List session recordings
- Download session videos (binary output)

### ☁️ Cloud Pipeline Ready
- Works with:
  - Backblaze B2 (S3-compatible)
  - AWS S3
  - Google Drive
- Optimized for large file handling

### ⚙️ Production Features
- Retry with exponential backoff
- Rate-limit safe (batch compatible)
- Error classification
- Pagination support
- Custom API endpoint (future-proof)
